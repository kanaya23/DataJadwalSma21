# Source Website dan Data Jadwal SMAN 21 Bekasi

Repositori ini berisi kode sumber website interaktif dan data jadwal pelajaran untuk SMA Negeri 21 Bekasi. Website ini memfasilitasi akses informasi jadwal pelajaran yang rapi, profesional, dan mudah digunakan baik di desktop maupun perangkat seluler.

## Fitur Website
- **Dua Shift Jadwal**: Mendukung Jadwal Shift Pagi (Kelas XI.5 - XI.8, XII) dan Shift Siang (Kelas X, XI.1 - XI.4).
- **Tampilan Fleksibel**:
  - Berdasarkan **Hari**: Menampilkan seluruh kelas untuk hari terpilih (otomatis beralih ke tampilan kartu di perangkat seluler).
  - Berdasarkan **Kelas**: Menampilkan jadwal mingguan lengkap untuk kelas terpilih.
  - Berdasarkan **Jadwal Piket**: Menampilkan jadwal piket guru, wakasek/staff, dan guru BK.
- **Tema Terang & Gelap**: Dilengkapi dengan tombol pengubah tema (Light/Dark Mode) yang snappy dan menyimpan preferensi pilihan.
- **Informasi Guru Instan**: Arahkan kursor (*hover*) pada kode pelajaran di desktop untuk menampilkan nama guru dan mata pelajaran secara instan.

## Kontribusi

Kami sangat menyambut kontribusi Anda untuk membantu pemeliharaan (*maintenance*) data jadwal pelajaran atau pengembangan fitur website. 

Anda dapat melakukan kontribusi dengan cara mengirimkan **Pull Request (PR)**:

1. **Fork** repositori ini ke akun GitHub Anda.
2. Buat branch baru untuk perubahan Anda (contoh: `git checkout -b update-jadwal-kelas-x`).
3. Lakukan perubahan data jadwal di `index.html` atau data terstruktur di `schedule_data.json`.
4. **Commit** perubahan Anda dengan pesan yang jelas (contoh: `git commit -m "Update jadwal kelas X.5 semester ganjil"`).
5. **Push** branch Anda ke repositori fork Anda (`git push origin update-jadwal-kelas-x`).
6. Kirimkan **Pull Request** dari repositori fork Anda ke repositori utama ini untuk ditinjau.
